### Added
- `generate_image_sd3` — SD3.5 image generation (Large, Large-Turbo, Medium)
- `generate_image_core` — Core image generation with style presets
- `generate_image_ultra` — Ultra image generation
- `remove_background` — Background removal
- `replace_background` — Background replacement with relighting
- `inpaint` — Region editing with mask
- `search_and_replace` — Object replacement without mask
- `outpaint` — Canvas extension in any direction
- `search_and_recolor` — Object recoloring without mask
- `upscale_fast` — 4x fast upscale
- `upscale_creative` — 4x creative upscale (async)
